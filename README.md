# University_work
このリポジトリには、大学での制作物をまとめています。主に「データサイエンス実験」という授業で行った実験レポートを収録しています。

## 📂 experiments フォルダについて
`experiments` フォルダには、授業内で行った各実験の内容が整理されています。  
各実験は以下の構成になっています：

- フォルダ内に **簡潔なまとめ（Markdownファイル）** と **詳細なレポート（PDFファイル）** を収録
- Markdownファイルは現在製作途中で、完成しているものは下記のみです
